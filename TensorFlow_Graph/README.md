## TensorFlow 计算图
Tensorflow 是一个通过计算图的形式来表述计算过程的编程系统，计算图也叫数据流图，它是一个有向无环图。Tensorflow 计算的过程就是利用 Tensor 来建立一个计算图，然后使用 Session 会话来启动计算，最后得到结果的过程。
